# Mealory Support

このリポジトリは、MealoryのApp Store申請用サポートページとプライバシーポリシーページを公開するためのものです。

Netlifyで公開している静的サイトです。

サポートページURL：
https://mealory-support.netlify.app

プライバシーポリシーURL：
https://mealory-support.netlify.app/privacy.html

問い合わせ先：
[mealory.contact@gmail.com](mailto:mealory.contact@gmail.com)

メールアドレスを変更する場合は、`index.html` と `privacy.html` の `mailto` 部分を編集してください。

このサイトはHTML/CSSのみで構成された静的サイトです。JavaScriptやバックエンドは使用していません。
